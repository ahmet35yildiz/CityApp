
# Super City App

An application that presents the weather and pharmacy information on duty according to the selected city, and provides a news feed regardless of the city.

## Project Features
• MVVM

• Dependency Injection - Hilt

• Retrofit

• Picasso

• RecyclerView

• Extensions

• Bottom Sheet

• Lottie Animations

• ViewPager - TabLayout

• Shared Preferences

• Splash Screen
